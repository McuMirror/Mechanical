Detailed Description

Contains the Attitude Control System (ACS) struct.

The ACS struct includes all the data needed to run the ACS. One struct should be created when the ACS starts. A pointer to the struct should then be passed to other ACS functions.

Created by Galen Savidge. Edited 5/16/2019.
