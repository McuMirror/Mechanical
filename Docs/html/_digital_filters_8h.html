Detailed Description

Filter sensor data.

Currently includes a moving average filter datatype, which runs a variable length digital moving average filter on a configurable number of sensor readings. Use newMovingAvgFilter() to allocate a MovingAvgFilter and then pass it to runMovingAvgFilter() each loop to filter sensor data.

Created by Galen Savidge. Edited 5/12/2019.
