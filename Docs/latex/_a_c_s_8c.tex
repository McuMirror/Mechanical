\subsection{Detailed Description}
Contains the Attitude Control System (A\+CS) struct. 

The A\+CS struct includes all the data needed to run the A\+CS. One struct should be created when the A\+CS starts. A pointer to the struct should then be passed to other A\+CS functions as appropriate.
