\subsection{Detailed Description}
Filter sensor data. 

Currently includes a moving average filter datatype, which runs a variable length digital moving average filter on a configurable number of sensor readings. Use \mbox{\hyperlink{_digital_filters_8h_a8ef38f79b0e457d8293ad5ecb350bd88}{new\+Moving\+Avg\+Filter()}} to allocate a Moving\+Avg\+Filter and then pass it to \mbox{\hyperlink{_digital_filters_8h_a7e8cbba6d231e26a3c97dcd5b4daf474}{run\+Moving\+Avg\+Filter()}} each loop to filter sensor data.
